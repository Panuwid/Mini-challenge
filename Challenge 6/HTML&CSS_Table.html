HTML&CSS_Table by P

Tag Name Description
<html> HTML กำหนดเอกสาร HTML
<head> Head ส่วนหัวของเอกสาร HTML (ไม่แสดงผล)
<title> Title กำหนดชื่อของหน้าเว็บ
<body> Body ส่วนเนื้อหาของเอกสาร HTML
<h1> ถึง <h6> Heading หัวข้อระดับ 1 ถึง 6
<p> Paragraph กำหนดย่อหน้า
<a> Anchor สร้างลิงก์
<img> Image แทรกรูปภาพ
<ul> Unordered List รายการแบบไม่เรียงลำดับ
<ol> Ordered List รายการแบบเรียงลำดับ
<li> List Item รายการย่อยในลิสต์
<div> Division กำหนดส่วน/บล็อกของเนื้อหา
<span> Span กำหนดส่วนของข้อความ
<br> Line Break ขึ้นบรรทัดใหม่
<table> Table สร้างตาราง

Attributes

Tag Name Description
id กำหนดตัวระบุเฉพาะสำหรับอีลิเมนต์ <div id="header">...</div>
class กำหนดคลาสสำหรับอีลิเมนต์ (ใช้ในการจัดรูปแบบ CSS) <p class="highlight">...</p>
style กำหนดรูปแบบ CSS แบบอินไลน์ <p style="color: blue;">...</p>
src ระบุแหล่งที่มาของไฟล์ (เช่น รูปภาพ, วิดีโอ, สคริปต์) <img src="image.jpg">
href ระบุ URL ของลิงก์ <a href="https://www.example.com">ลิงก์</a>
alt กำหนดข้อความทางเลือกสำหรับรูปภาพ <img src="logo.png" alt="โลโก้บริษัท">
title กำหนดข้อมูลเพิ่มเติมเมื่อเลื่อนเมาส์ไปชี้ <a href="#" title="คลิกเพื่อดูรายละเอียด">ข้อมูลเพิ่มเติม</a>
target ระบุวิธีการเปิดลิงก์ (เช่น หน้าต่างใหม่) <a href="page.html" target="_blank">เปิดในหน้าต่างใหม่</a>
name กำหนดชื่อให้กับอีลิเมนต์ (ใช้บ่อยกับฟอร์ม) <input type="text" name="username">
value กำหนดค่าเริ่มต้นให้กับอีลิเมนต์ input <input type="text" value="ค่าเริ่มต้น">
placeholder กำหนดข้อความตัวอย่างในช่องกรอกข้อมูล <input type="text" placeholder="กรุณากรอกชื่อ">
disabled ทำให้อีลิเมนต์ไม่สามารถใช้งานได้ <button disabled>ปุ่มที่ถูกปิดการใช้งาน</button>
readonly ทำให้อีลิเมนต์อ่านได้อย่างเดียว ไม่สามารถแก้ไขได้ <input type="text" value="ข้อมูลที่แก้ไขไม่ได้" readonly>
required กำหนดให้ต้องกรอกข้อมูลในฟิลด์นั้นๆ <input type="text" required>

CSS

Tag Name Description
color กำหนดสีของข้อความ color: #ff0000;
background-color กำหนดสีพื้นหลัง background-color: #f0f0f0;
font-size กำหนดขนาดตัวอักษร font-size: 16px;
font-family กำหนดแบบอักษร font-family: Arial, sans-serif;
font-weight กำหนดความหนาของตัวอักษร font-weight: bold;
text-align กำหนดการจัดวางแนวข้อความ text-align: center;
margin กำหนดระยะขอบนอก margin: 10px 5px;
padding กำหนดระยะขอบใน padding: 15px;
width กำหนดความกว้าง width: 100%;
height กำหนดความสูง height: 200px;
display กำหนดลักษณะการแสดงผล display: flex;
position กำหนดตำแหน่งของอีลิเมนต์ position: absolute;
border กำหนดเส้นขอบ border: 1px solid #000;
border-radius กำหนดความโค้งของมุม border-radius: 5px;
box-shadow กำหนดเงาให้กับอีลิเมนต์ box-shadow: 2px 2px 5px #888;
opacity กำหนดความโปร่งใส opacity: 0.8;
transition กำหนดการเปลี่ยนแปลงแบบนุ่มนวล transition: all 0.3s ease;
z-index กำหนดลำดับการซ้อนทับ z-index: 100;
